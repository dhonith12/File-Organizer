# 📂 File Organizer

A simple **File Organizer** that automatically sorts files into folders based on their file types such as Images, Videos, Documents, and more.  
This helps keep directories clean and organized without manual effort.

---

## 🚀 Live Demo
Access the project here:

🔗 https://dhonith12.github.io/File-Organizer/

---

## 📌 Features
- Automatically detects file extensions
- Moves files into categorized folders
- Improves file management efficiency
- Lightweight and easy to run
- HTML interface provides a downloadable ZIP of organized files
